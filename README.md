# TMB July 2026

Static website for the July 2026 Tour du Mont Blanc trip.

## Source

Built from `TMB-July2026-Itinerary.pdf`. Private booking and reservation numbers are intentionally omitted from the website.

## Local Preview

Open `index.html` directly in a browser, or serve the folder with any static file server.

## Trip Arc

- July 1-2: Seattle to Geneva
- July 4: Chamonix / Les Houches arrival
- July 5-11: 7-day guided Tour du Mont Blanc
- July 12: extra Chamonix day
- July 13-14: Zermatt, Gornergrat, Riffelsee, Matterhorn Glacier Paradise
- July 15: scenic rail to Zurich via Visp, Andermatt, and Lucerne
- July 16: Zurich to Seattle
